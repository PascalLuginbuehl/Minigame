What is this?

A 2D multiplayer Minigame written in TypeScript from scratch

Installation

Install node.js version >= 6.6.0

Run as Admin

$ npm install gulp-cli -g

Run Webserver

Navigate into Minigame folder This will install depencies and run local express webserver

> npm start

Connect with a ECMAScript 2015 compatible browser to localhost (Chrome 53, Edge 12...)
